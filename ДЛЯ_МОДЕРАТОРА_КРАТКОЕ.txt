ОБЪЯСНЕНИЕ НЕОБХОДИМОСТИ РАЗРЕШЕНИЙ

Приложение: Bluetooth111
Назначение: Управление электронными устройствами через Bluetooth (Arduino, ESP32)

════════════════════════════════════════════════════════════════

1. BLUETOOTH_CONNECT (Android 12+)
Назначение: Установка соединения с устройствами Bluetooth
Использование: Подключение к Arduino/ESP32 для отправки команд управления
Без этого: Основной функционал приложения не работает

2. BLUETOOTH_SCAN (Android 12+)
Назначение: Поиск доступных Bluetooth устройств
Использование: Отображение списка устройств для подключения
Без этого: Пользователь не сможет найти свои устройства

3. ACCESS_FINE_LOCATION
4. ACCESS_COARSE_LOCATION
Назначение: Обязательное требование Android для Bluetooth сканирования на Android 6-11
Использование: ТОЛЬКО для работы Bluetooth API (требование системы)
Важно: НЕ используется для определения местоположения пользователя
        НЕ отправляется на сервер
        Данные геолокации НЕ собираются
Без этого: На Android 6-11 сканирование Bluetooth не работает

════════════════════════════════════════════════════════════════

ГАРАНТИИ:
✓ Местоположение пользователя НЕ определяется
✓ Данные НЕ передаются на сервер
✓ Все работает локально на устройстве
✓ Разрешения используются только для Bluetooth
✓ Соответствует требованиям Google Play

Документация Android: https://developer.android.com/guide/topics/connectivity/bluetooth/permissions
