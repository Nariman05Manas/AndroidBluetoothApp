КОПИРУЙТЕ ТЕКСТ НИЖЕ В ФОРМУ МОДЕРАЦИИ
═══════════════════════════════════════════════════════════════

▼ ACCESS_COARSE_LOCATION
═══════════════════════════════════════════════════════════════
Обязательное требование Android для сканирования Bluetooth устройств на версиях Android 6-11. Используется ТОЛЬКО для работы Bluetooth API. НЕ используется для определения местоположения пользователя. На Android 12+ это разрешение не используется (заменено на BLUETOOTH_SCAN).
Документация: https://developer.android.com/develop/connectivity/bluetooth/bt-permissions#declare-android11-or-lower


▼ ACCESS_FINE_LOCATION
═══════════════════════════════════════════════════════════════
Обязательное требование Android для сканирования Bluetooth устройств на версиях Android 6-11. Используется ТОЛЬКО для работы Bluetooth API. НЕ используется для определения местоположения пользователя. Координаты GPS НЕ запрашиваются и НЕ отправляются на сервер. На Android 12+ это разрешение не используется (заменено на BLUETOOTH_SCAN).
Документация: https://developer.android.com/develop/connectivity/bluetooth/bt-permissions#declare-android11-or-lower


▼ BLUETOOTH_CONNECT
═══════════════════════════════════════════════════════════════
Необходимо для подключения к устройствам Bluetooth (Arduino, ESP32, микроконтроллеры). Основной функционал приложения - отправка команд управления моторами и сервоприводами через Bluetooth. Без этого разрешения приложение не сможет выполнять свою основную функцию.
Документация: https://developer.android.com/guide/topics/connectivity/bluetooth/permissions


▼ BLUETOOTH_SCAN
═══════════════════════════════════════════════════════════════
Необходимо для поиска доступных Bluetooth устройств. Приложение отображает список найденных устройств, чтобы пользователь мог выбрать и подключиться к своему Arduino или ESP32. Без этого разрешения пользователь не сможет найти свои устройства для подключения.
Документация: https://developer.android.com/guide/topics/connectivity/bluetooth/permissions

ВАЖНО ДЛЯ МОДЕРАТОРА:
═══════════════════════════════════════════════════════════════

✓ Приложение НЕ собирает геолокацию пользователя
✓ Разрешения LOCATION используются только для Bluetooth API
✓ Данные НЕ передаются на внешние серверы
✓ Все работает локально на устройстве пользователя
